# 🛒 Zepto Product Inventory Analysis

## 📌 Overview

This project analyzes Zepto's product inventory dataset to derive meaningful business insights using **MySQL** and presents them through an interactive **Power BI dashboard**. The workflow includes data cleaning, SQL-based analysis, dashboard development, report writing, and presentation creation using **Gamma**.

The project demonstrates an end-to-end data analytics workflow—from raw data to business-ready insights.

---

## 📂 Dataset

The dataset contains product-level inventory information from Zepto, including:

* Product Name
* Category
* MRP
* Selling Price
* Discount Percentage
* Available Quantity
* Product Weight
* Stock Availability

The data was cleaned and transformed before performing analysis.

## 🔄 Project Workflow

### 1. Data Cleaning (MySQL)

* Checked for missing and duplicate records
* Removed invalid entries (e.g., zero MRP)
* Converted price values to a consistent format
* Standardized the dataset for analysis

### 2. SQL Analysis

Performed business-focused analysis to answer questions such as:

* Which are the Top 10 most expensive products?
* How are products distributed across different discount percentages?
* Which categories have the highest number of out-of-stock products?
* Which categories contribute the highest inventory and potential sales value?
* How does pricing vary across different product categories?

### 3. Dashboard Development (Power BI)

Built an interactive dashboard featuring:

* Top 10 Expensive Products
* Quantity by Discount %
* Out-of-Stock Products by Category
* Category-wise Inventory Summary
* Selling Price Slicer for dynamic filtering

### 4. Documentation & Presentation

* Prepared a detailed project report summarizing the analysis and business insights.
* Designed a professional presentation using Gamma to showcase the project workflow and findings.

## 🚀 How to Run

### SQL Analysis

1. Open **MySQL Workbench**.
2. Import the Zepto dataset.
3. Execute the SQL script to perform data cleaning and analysis.

### Power BI Dashboard

1. Open the `.pbix` file in **Power BI Desktop**.
2. Refresh the data source if required.
3. Explore the dashboard using the Selling Price slicer.

### Report & Presentation

* Open the project report (`.docx` or `.pdf`) for detailed documentation.
* Open the Gamma presentation to review the project summary and key insights.
